# Upstream Team Skills

这个目录包含 Upstream 团队（上游开发贡献团队）专属的 skills。

## 团队职责

Upstream 团队负责贡献上游开源项目，包括：
- 向上游开源项目提交 PR 和 Patch
- 参与上游社区讨论和决策
- 跟踪上游技术演进
- 将上游最新特性集成到项目中
- 维护与上游社区的良好关系
